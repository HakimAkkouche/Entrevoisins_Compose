EntreVoisins (Kotlin version

# Sujets abordés / démontrés
 * Kotlin
 * Architecture MVVM (Model View ViewModel)
 * `LiveData` 
 * `Coroutine` & `Flow`
 * `Jetpack Compose`
 * `RoomDatabase` `Repository` & `Dao Room`
